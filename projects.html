Current Projects

Some tools disrupt. Some tools create. I use both. These are my current operations.

  • Java Library Management System — A full-stack desktop application built in Java to manage book inventories, user records, and lending operations. Designed with object-oriented principles and clean UI logic, it streamlines library workflows with speed and precision.
  • WiFi Deauthentication Tool — A Kali Linux-based terminal utility that exploits 802.11 protocol vulnerabilities to send deauth packets and disrupt active WiFi connections. Built for educational and testing purposes, it demonstrates low-level network manipulation and command-line control.
